# ⚡ 빠른 시작 가이드

## 🎯 30초 안에 시작하기

### 1. 프로젝트 폴더 열기
```
Windows 탐색기 → C:\Git\Routine_app
```

### 2. 원클릭 실행
```
quick_start.bat 더블클릭
```

### 3. 화면 지시 따르기
```
Enter 키 → 1번 선택 → Enter 키
```

### 4. PyCharm 확인
```
7개 터미널 탭이 자동으로 생성됨!
```
